# Utils Directory

Helper scripts supporting some of the slash commands in this configuration.

## Contents

- **git-rewrite-commit-descriptions-helpers/** - Shell helpers used by `/git:rewrite-commit-descriptions` (detect bad commits, analyze, generate, and apply rewrites). See that folder's own README for details.
- **iterm-paste.sh** - Sends keystrokes to the iTerm2 session that launched it.
- **nostop_state/** - Scratch state directory used at runtime.
